# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

Everyone interacting in this project's spaces — issues, pull requests, code,
documentation, and any other channel — is expected to follow it.

## Reporting

Report unacceptable behavior to `conduct@noix.dev`. Reports are handled
confidentially. Maintainers will review and respond as quickly as possible.
